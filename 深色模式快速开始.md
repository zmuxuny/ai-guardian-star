## 🔧 为新页面添加主题支持

### 步骤 1: 导入主题管理器
```typescript
import { ThemeManager } from '../common/ThemeManager';
```

### 步骤 2: 添加主题实例
```typescript
@Component
export struct YourPage {
  @State themeManager: ThemeManager = ThemeManager.getInstance();
  
  // 其他代码...
}
```

### 步骤 3: 使用主题颜色
```typescript
// 文字颜色
Text('标题')
  .fontColor(this.themeManager.colors.primaryTextColor)

// 背景颜色
Column()
  .backgroundColor(this.themeManager.colors.backgroundColor)

// 卡片背景
Row()
  .backgroundColor(this.themeManager.colors.cardBackgroundColor)
```

### 完整示例
```typescript
import { ThemeManager } from '../common/ThemeManager';

@Component
export struct MyPage {
  @State themeManager: ThemeManager = ThemeManager.getInstance();
  
  build() {
    Column() {
      // 标题
      Text('页面标题')
        .fontSize(20)
        .fontColor(this.themeManager.colors.primaryTextColor)
      
      // 卡片
      Column() {
        Text('卡片内容')
          .fontColor(this.themeManager.colors.primaryTextColor)
        
        Text('副标题')
          .fontColor(this.themeManager.colors.secondaryTextColor)
      }
      .backgroundColor(this.themeManager.colors.cardBackgroundColor)
      .padding(16)
      .borderRadius(12)
    }
    .width('100%')
    .height('100%')
    .backgroundColor(this.themeManager.colors.backgroundColor)
  }
}
```

## 🎨 常用颜色映射表

| 场景 | 颜色变量 | 浅色模式 | 深色模式 |
|------|----------|---------|---------|
| 页面背景 | `backgroundColor` | 白色 | 黑色 |
| 主要文字 | `primaryTextColor` | 深黑 | 白色 |
| 次要文字 | `secondaryTextColor` | 灰色 | 浅白 |
| 辅助文字 | `tertiaryTextColor` | 浅灰 | 深灰 |
| 卡片背景 | `cardBackgroundColor` | 白色 | 深灰 |
| 主按钮 | `primaryButtonColor` | 蓝色 | 亮蓝 |
| 危险按钮 | `dangerButtonColor` | 红色 | 亮红 |
| 分割线 | `separatorColor` | 浅灰 | 深灰 |

## 💡 实用技巧

### 1. 调试主题
查看主题相关日志：
```bash
# DevEco Studio 日志面板搜索
[Theme]
```

### 2. 判断当前主题
```typescript
if (this.themeManager.isDarkMode) {
  // 深色模式特殊处理
} else {
  // 浅色模式特殊处理
}
```

### 3. 手动切换主题
```typescript
// 切换主题
this.themeManager.toggleTheme();

// 设置为深色
this.themeManager.setTheme(true);

// 设置为浅色
this.themeManager.setTheme(false);
```

### 4. 特殊颜色处理
某些颜色需要在两种模式下使用不同值：
```typescript
.backgroundColor(
  this.themeManager.isDarkMode ? '#3D1F1F' : '#FFF5F5'
)
```

## ⚠️ 注意事项

1. **避免硬编码颜色**: 始终使用 `themeManager.colors.*`
2. **测试两种模式**: 确保在浅色和深色模式下都清晰
3. **图标颜色**: 使用 `.fillColor()` 应用主题颜色
4. **品牌色例外**: 警告红色等品牌色可保持不变
