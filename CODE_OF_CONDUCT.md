# 行为守则（Code of Conduct）

本项目采用 Contributor Covenant 行为守则 (版本 2.1)。我们致力于为参与者提供一个友好、包容和尊重的社区环境。所有参与者（包括贡献者、维护者、提交 issue 或 PR 的人）都应遵守以下规范：

- 尊重他人，不发表人身攻击、歧视或骚扰性言论。
- 避免使用攻击性或侮辱性的语言，尊重不同观点与文化差异。
- 在讨论中关注事实和问题本身，避免人身攻击。

若你遭遇或目睹任何不当行为，请联系维护者并提供相关证据（例如链接、截图）。

维护者将尽力调查并在必要时采取行动，包括警告、移除内容或禁止参与者等措施。

（此处可替换为 Contributor Covenant 官方英文全文或链接。）
