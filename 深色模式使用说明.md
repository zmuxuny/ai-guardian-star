# 深色模式使用说明

## 功能概述

应用已集成完整的深色模式适配功能，支持用户在浅色和深色两种主题之间切换。

## 主要特性

### 1. 全局主题管理
- 使用单例模式的 `ThemeManager` 类管理全局主题
- 支持响应式主题切换
- 主题偏好自动保存到本地存储

### 2. 主题颜色
深色模式和浅色模式都定义了完整的颜色体系：

#### 浅色模式
- 背景色：#FFFFFF (白色)
- 次级背景色：#F2F2F7 (浅灰)
- 卡片背景色：#FFFFFF (白色)
- 主文字色：#1C1C1E (深黑)
- 次级文字色：#8E8E93 (灰色)
- 分割线色：#E5E5EA (浅灰)

#### 深色模式
- 背景色：#000000 (黑色)
- 次级背景色：#1C1C1E (深灰)
- 卡片背景色：#2C2C2E (深灰)
- 主文字色：#FFFFFF (白色)
- 次级文字色：#EBEBF5 (浅白)
- 分割线色：#38383A (中灰)

### 3. 本地存储
- 主题偏好自动保存
- 应用重启后保持用户选择的主题

### 4. 状态栏适配
- 深色模式下状态栏文字为白色
- 浅色模式下状态栏文字为黑色

## 使用方法

### 在现有页面中使用主题

1. **导入 ThemeManager**
```typescript
import { ThemeManager } from '../common/ThemeManager';
```

2. **在组件中获取主题实例**
```typescript
@Component
export struct YourPage {
  @State themeManager: ThemeManager = ThemeManager.getInstance();
  
  build() {
    Column() {
      Text('示例文本')
        .fontColor(this.themeManager.colors.primaryTextColor)
    }
    .backgroundColor(this.themeManager.colors.backgroundColor)
  }
}
```

3. **使用主题颜色**
```typescript
// 文字颜色
.fontColor(this.themeManager.colors.primaryTextColor)    // 主文字
.fontColor(this.themeManager.colors.secondaryTextColor)  // 次级文字
.fontColor(this.themeManager.colors.tertiaryTextColor)   // 三级文字

// 背景颜色
.backgroundColor(this.themeManager.colors.backgroundColor)         // 页面背景
.backgroundColor(this.themeManager.colors.cardBackgroundColor)     // 卡片背景
.backgroundColor(this.themeManager.colors.secondaryBackgroundColor) // 次级背景

// 按钮颜色
.backgroundColor(this.themeManager.colors.primaryButtonColor)  // 主按钮
.backgroundColor(this.themeManager.colors.dangerButtonColor)   // 危险按钮

// 边框和分割线
.border({ color: this.themeManager.colors.borderColor })
.divider({ strokeWidth: 1, color: this.themeManager.colors.separatorColor })
```

### 切换主题

在个人中心页面，用户可以通过开关切换主题：

```typescript
Toggle({ type: ToggleType.Switch, isOn: this.themeManager.isDarkMode })
  .onChange((isOn: boolean) => {
    this.themeManager.setTheme(isOn);
  })
```

### 手动切换主题

在代码中可以手动切换主题：

```typescript
// 切换主题
ThemeManager.getInstance().toggleTheme();

// 设置为深色模式
ThemeManager.getInstance().setTheme(true);

// 设置为浅色模式
ThemeManager.getInstance().setTheme(false);
```

## 适配步骤

为新页面添加深色模式支持的步骤：

1. **导入 ThemeManager**
2. **添加主题管理器实例**
3. **替换所有硬编码的颜色值**
   - 将 `Color.White` 改为 `this.themeManager.colors.backgroundColor`
   - 将 `Color.Black` 改为 `this.themeManager.colors.primaryTextColor`
   - 将其他颜色替换为对应的主题颜色
4. **测试两种模式下的显示效果**

## 示例代码

完整的页面适配示例：

```typescript
import { ThemeManager } from '../common/ThemeManager';

@Component
export struct ExamplePage {
  @State themeManager: ThemeManager = ThemeManager.getInstance();
  
  build() {
    Column() {
      // 页面标题
      Text('页面标题')
        .fontSize(20)
        .fontWeight(FontWeight.Bold)
        .fontColor(this.themeManager.colors.primaryTextColor)
        .margin({ top: 20, bottom: 20 })
      
      // 卡片容器
      Column() {
        Text('卡片内容')
          .fontSize(16)
          .fontColor(this.themeManager.colors.primaryTextColor)
        
        Text('描述文字')
          .fontSize(14)
          .fontColor(this.themeManager.colors.secondaryTextColor)
      }
      .width('90%')
      .padding(16)
      .backgroundColor(this.themeManager.colors.cardBackgroundColor)
      .borderRadius(12)
      
      // 按钮
      Button('确定')
        .backgroundColor(this.themeManager.colors.primaryButtonColor)
        .fontColor(Color.White)
    }
    .width('100%')
    .height('100%')
    .backgroundColor(this.themeManager.colors.backgroundColor)
  }
}
```

## 注意事项

1. **响应式更新**：使用 `@State themeManager` 确保主题切换时 UI 自动更新
2. **颜色一致性**：尽量使用主题预定义的颜色，避免硬编码
3. **对比度**：确保文字和背景有足够的对比度
4. **图标颜色**：使用 `.fillColor()` 为图标应用主题颜色
5. **测试**：在两种模式下都要测试页面显示效果

## 调试

查看主题相关日志：
```

```

## 技术实现

- **单例模式**：全局唯一的主题管理器实例
- **响应式状态**：使用 `@Observed` 和 `@Track` 实现响应式更新
- **本地存储**：使用 `@ohos.data.preferences` 持久化主题偏好
- **系统 UI 适配**：使用 `window.setWindowSystemBarProperties` 更新状态栏
