# 图标自适应说明

> 图标方案经历两个阶段：1) SvgIcon 组件自适应 → 2) Image 替换 SvgIcon（详见 `UI重构更新说明.md`）

## 自适应策略（第一阶段：SvgIcon 组件）

### 基准尺寸（以 360px 屏幕为基准）

| 类型 | 基准尺寸 | 适配范围 | 使用场景 |
|------|---------|---------|---------|
| `nav` | 32px | 27–37px | 底部导航图标 |
| `list` | 26px | 22–30px | 列表项图标 |
| `card` | 32px | 27–37px | 卡片状态图标 |
| `empty` | 56px | 48–64px | 空状态大图标 |
| `header` | 36px | 31–41px | 页面标题旁图标 |
| `custom` | 24px | 固定值 | 自定义尺寸 |

### 适配算法

```typescript
const scaleFactor = screenWidth / 360;          // 缩放系数
const clampedFactor = Math.max(0.85, Math.min(1.15, scaleFactor)); // ±15% 限幅
return Math.round(baseSize * clampedFactor);    // 最终尺寸
```

### 尺寸调整历史（2026-03-17）

| 类型 | 原尺寸 | 新尺寸 | 增幅 | 使用场景 |
|------|-------|-------|------|---------|
| `nav` | 32px | 40px | +25% | 底部导航图标 |
| `list` | 26px | 34px | +31% | 列表项图标 |
| `card` | 32px | 40px | +25% | 卡片状态图标 |
| `empty` | 56px | 72px | +29% | 空状态大图标 |
| `header` | 36px | 48px | +33% | 页面标题旁图标 |

线条粗细：`strokeWidth` 从 1.8 → 2.2

### 不同设备上的实际尺寸

| 设备类型 | 屏幕宽度 | nav | list | card | empty | header |
|---------|---------|-----|------|------|-------|--------|
| 小屏手机 | 320px | 34px | 29px | 34px | 61px | 41px |
| 标准手机 | 360px | 40px | 34px | 40px | 72px | 48px |
| 大屏手机 | 412px | 46px | 39px | 46px | 83px | 55px |
| 平板 | 720px | 46px* | 39px* | 46px* | 83px* | 55px* |

\*平板因 ±15% 限幅，使用最大允许值

## 第二阶段：Image 替换 SvgIcon

`UI重构更新说明.md` 记录了 2026-03-18 的全面重构：图标从 SvgIcon 组件改为 Image 加载独立 SVG，颜色烘焙进文件。当前项目使用此方案。
