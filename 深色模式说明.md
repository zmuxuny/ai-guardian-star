# 深色模式说明

## 功能概述

应用已集成完整的深色模式适配功能，支持用户在浅色和深色两种主题之间切换。使用单例模式 `ThemeManager` 管理全局主题，支持响应式切换，偏好自动保存到本地存储。

## 快速上手

### 切换主题
1. 启动应用并登录
2. 点击底部「个人」Tab
3. 找到「深色模式」开关并点击，页面立即切换主题
4. 完全关闭应用后重启，主题偏好自动保持

## 主题颜色体系

### 浅色模式
- 背景色：`#FFFFFF` (白色)
- 次级背景色：`#F2F2F7` (浅灰)
- 卡片背景色：`#FFFFFF` (白色)
- 主文字色：`#1C1C1E` (深黑)
- 次级文字色：`#8E8E93` (灰色)
- 分割线色：`#E5E5EA` (浅灰)

### 深色模式
- 背景色：`#000000` (黑色)
- 次级背景色：`#1C1C1E` (深灰)
- 卡片背景色：`#2C2C2E` (深灰)
- 主文字色：`#FFFFFF` (白色)
- 次级文字色：`#EBEBF5` (浅白)
- 分割线色：`#38383A` (中灰)

## 为新页面添加主题支持

### 步骤 1：导入主题管理器
```typescript
import { ThemeManager } from '../common/ThemeManager';
```

### 步骤 2：在组件中获取实例
```typescript
@Component
export struct YourPage {
  @State themeManager: ThemeManager = ThemeManager.getInstance();
  // ...
}
```

### 步骤 3：使用主题颜色
```typescript
// 文字颜色
.fontColor(this.themeManager.colors.primaryTextColor)    // 主文字
.fontColor(this.themeManager.colors.secondaryTextColor)  // 次级文字

// 背景颜色
.backgroundColor(this.themeManager.colors.backgroundColor)         // 页面背景
.backgroundColor(this.themeManager.colors.cardBackgroundColor)     // 卡片背景

// 按钮颜色
.backgroundColor(this.themeManager.colors.primaryButtonColor)  // 主按钮
.backgroundColor(this.themeManager.colors.dangerButtonColor)   // 危险按钮

// 边框和分割线
.border({ color: this.themeManager.colors.borderColor })
```

### 常用颜色映射表

| 场景 | 变量 | 浅色 | 深色 |
|------|------|------|------|
| 页面背景 | `backgroundColor` | #FFFFFF | #000000 |
| 主要文字 | `primaryTextColor` | #1C1C1E | #FFFFFF |
| 次要文字 | `secondaryTextColor` | #8E8E93 | #EBEBF5 |
| 卡片背景 | `cardBackgroundColor` | #FFFFFF | #2C2C2E |
| 主按钮 | `primaryButtonColor` | 蓝色 | 亮蓝 |
| 危险按钮 | `dangerButtonColor` | 红色 | 亮红 |

### 条件颜色处理
```typescript
// 需要同时考虑功能状态和主题时
```

## 技术实现

- **单例模式**：全局唯一 `ThemeManager` 实例
- **响应式状态**：`@Observed` + `@Track` 实现自动 UI 更新
- **本地持久化**：`@ohos.data.preferences` 保存主题偏好
- **系统 UI 适配**：`window.setWindowSystemBarProperties` 更新状态栏颜色

### 工作流程
```
用户切换开关 → ThemeManager.setTheme(isDark)
  → 更新 isDarkMode 状态 → 应用新主题颜色
  → 保存到本地存储 → 更新系统状态栏
  → 所有页面自动重新渲染
```

## 注意事项

1. **避免硬编码颜色**：始终使用 `themeManager.colors.*`
2. **测试两种模式**：确保浅色和深色下都清晰可读
3. **图标颜色**：使用 `.fillColor()` 应用主题颜色
4. **品牌色例外**：警告红色等可保持不变
5. **调试日志**：DevEco Studio 搜索 `[Theme]` 标签
